# Desafio de Projeto
## Criando modelos com Python e Machine Learning para prever a evolução do COVID-19 no Brasil
## Bootcamp: **Geração Tech Unimed-BH - Ciência de Dados**
### DIO.me

### DESCRIÇÃO

Neste Labs, vamos investigar os dados da disseminação do COVID-19 no Brasil desde o início da pandemia em fevereiro de 2020. 
Iremos estudar os padrões de disseminação da doença, investigar os números de infectados, de recuperados e de óbitos, para 
construir modelos com Python e Machine Learning para prever os números nos próximos dias e o ponto de virada da curva de infecção 
baseado em alguns cenários.

Foi instalado o Anaconda com o Python 3.9 no Windows mas foi necessário criar um ambiente com o Python 3.7 para rodar a biblioteca de predição do Facebook fbprophet.
